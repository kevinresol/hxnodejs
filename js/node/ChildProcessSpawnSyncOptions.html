package js.node

typedefChildProcessSpawnSyncOptions

import js.node.ChildProcess

Options for the spawnSync method.

Properties

cwd:Null<String>

Current working directory of the child process.

encoding:Null<String>

Default: 'utf8'

env:Null<DynamicAccess<String>>

Environment key-value pairs

gid:Null<Int>

Sets the group identity of the process. See setgid(2).

input:Null<EitherType<String, Buffer>>

killSignal:Null<String>

The child process is killed with killSignal (default: 'SIGTERM').

maxBuffer:Null<Int>

The largest amount of data allowed on stdout or stderr. If this value is exceeded then the child process is killed. Default: 200*1024

stdio:Null<ChildProcessSpawnOptionsStdio>

Child's stdio configuration.

timeout:Null<Int>

If greater than 0, then it will kill the child process if it runs longer than timeout milliseconds.

uid:Null<Int>

Sets the user identity of the process. See setuid(2).
