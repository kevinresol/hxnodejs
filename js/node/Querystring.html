package js.node

classQuerystring

This module provides utilities for dealing with query strings.

Static methods

staticdynamic escape (obj:Dynamic):String

The escape function used by Querystring.stringify, provided so that it could be overridden if necessary.

staticparse (str:String, ?sep:String, ?eq:String):DynamicAccess<String>

staticparse (str:String, ?options:{maxKeys:Int}):DynamicAccess<String>

staticparse (str:String, sep:String, ?options:{maxKeys:Int}):DynamicAccess<String>

Deserialize a query string to an object. Optionally override the default separator ('&') and assignment ('=') characters.

Options object may contain maxKeys property (equal to 1000 by default), it'll be used to limit processed keys. Set it to 0 to remove key count limitation.

staticstringify (obj:{}, ?sep:String, ?eq:String):String

Serialize an object to a query string. Optionally override the default separator ('&') and assignment ('=') characters.

staticdynamic unescape (str:String):Dynamic

The unescape function used by Querystring.parse, provided so that it could be overridden if necessary.
